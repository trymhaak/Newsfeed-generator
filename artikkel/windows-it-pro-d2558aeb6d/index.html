Kontekstbaserte omdirigeringer i Windows 365 nå i offentlig forhåndsvisning

Windows 365 får kontekstbaserte omdirigeringer i offentlig forhåndsvisning, som lar admins styre enhets- og ressursomdirigering basert på signaler som compliance-status, gruppemedlemskap og nettverksforhold. Funksjonen bruker Entra betinget tilgang sammen med Windows 365 og Azure Virtual Desktop for å tilpasse tilgang etter sesjonens tillitsnivå. Dette styrker BYOD-sikkerheten ved å begrense risikofylte dataveier på uadministrerte enheter.
